# 課程通知機器人(course-notify-line-bot)
提供課程提醒、每日天氣資訊、1A2B小遊戲 ，能自動抓取CSV檔內的上課時間，並於上課前30分鐘主動推播給使用者；輸入“查詢天氣 XX市 ”可查詢天氣 ；輸入“1A2B遊戲”可進行遊戲。
